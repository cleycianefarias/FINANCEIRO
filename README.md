API de controle financeiro que gerencia pessoas, categorias e transações, com geração de relatórios de receitas, despesas e saldo.

## 🛠 Tecnologias Utilizadas

### 🔙 Backend
- **.NET 8 / ASP.NET Core Web API** → construção da API
- **Entity Framework Core** → acesso e manipulação do banco de dados
- **LINQ** → consultas e agregações de dados
- **DataAnnotations** → validação de dados
- **Arquitetura em camadas (Controller + Service)** → organização do código

### 🎨 Frontend
- **React** → construção da interface
- **TypeScript** → tipagem e maior segurança no código
- **Fetch API** → comunicação com o backend
- **Hooks (useState, useEffect)** → gerenciamento de estado e ciclo de vida
